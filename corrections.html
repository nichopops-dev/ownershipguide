Corrections policy

How to request a correction

  • Send the page URL and what you believe is wrong via our contact page.
  • If you have a source, include it (official page, statute, government announcement, bank fee schedule, etc.).
  • If it’s a calculator issue, include your inputs and the output you expected (screenshots help).

What happens next

  • Confirm: we reproduce the issue and check against primary sources where possible.
  • Fix: we update the page and/or model assumptions.
  • Disclose: we note meaningful changes in the page’s “Last updated” line and keep changes transparent.

What counts as a correction

  • Wrong numbers, wrong formulas, or missing costs in a model.
  • Outdated rules/fees (e.g., policy changes from IRAS/HDB/LTA/MAS/CPF).
  • Broken links or misleading phrasing that changes a decision outcome.

Our goal

OwnershipGuide.com publishes simplified decision models for Singapore (calculators, comparisons, and guides). We aim to be clear about assumptions and to correct mistakes quickly.

What counts as a correction

  • Wrong figures, formulas, or logic in a calculator/model
  • Outdated rules, fees, limits, or definitions
  • Broken internal links or misleading navigation
  • Typos that change meaning

How to report an issue

Send us the page URL, a short description of the issue, and (if relevant) the source you’re referencing. Use the contact page.

  • Best: “URL + what looks wrong + what it should be”
  • Helpful: screenshot, steps to reproduce, or sample inputs

How we handle updates

  • Fix first: we prioritise correctness over wording.
  • Document assumptions: if a model uses simplifications, we state them.
  • Date stamp: key pages show a “Last updated” date near the end of the article.

Note: calculators are not quotes and do not replace professional advice. Always verify against official terms and the latest published rules.
