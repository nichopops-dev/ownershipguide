Corrections policy

OwnershipGuide.com is a planning tool. If you spot an error, we want to fix it quickly and transparently.

What counts as a correction

  • Wrong numbers or logic in a calculator/model (formulas, assumptions, missing cost items).
  • Outdated rules, fees, limits, or definitions (e.g., changes from IRAS, HDB, CPF, MAS, LTA).
  • Broken internal links or navigation issues that block a user from finding the right page.
  • Typos that change meaning (not cosmetic spelling).

How to request a correction

Please send the following via the contact page:

  • The page URL
  • What looks wrong and what you believe it should be
  • A source link (official page preferred) if the issue is rule/fee related
  • If it’s a calculator issue: the inputs you used and the output you saw (a screenshot helps)

What happens next

  • We’ll confirm we received your report (usually within a few business days).
  • If it’s clearly incorrect, we’ll update the page and reflect it in the page’s “Last updated”.
  • If it’s an assumption/edge-case issue, we may clarify assumptions instead of changing the result.

How we handle updates

  • We update pages when rules/fees change or when we find mistakes.
  • For modelling pages, we keep models simple on purpose. Assumptions and limitations are documented.
  • Quotes vary by provider and eligibility. Verify with official sources before committing to high-cost decisions.
