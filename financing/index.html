How to use the financing hub

This hub is a practical map of the “money layer” behind big Singapore decisions — mortgages, affordability constraints, and common trade-offs like fixed vs floating rates. Use it when you want to move from opinions to a simple model you can sanity-check.

  • Start with constraints: understand TDSR/MSR, cash/CPF buffers, and how rate changes affect your monthly downside.
  • Then compare options: refinance vs reprice, fixed vs floating, or “pay down mortgage vs invest”.
  • Finally run calculators: use stress tests and amortisation models to see whether the decision is fragile.

We publish simplified decision models (not quotes). When rules or typical market terms change, we refresh the pages and note assumptions.
