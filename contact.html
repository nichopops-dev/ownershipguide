We welcome feedback, corrections, and thoughtful suggestions. If something feels confusing or inaccurate, we want to know — we ship versioned updates and keep references where primary sources exist.

Email

You can reach us at contact@ownershipguide.com.

What you can contact us about

  • Corrections: broken links, incorrect assumptions, outdated rates, or calculation edge cases.
  • Feedback: which tools you want next, what’s confusing, what would make decisions easier.
  • Partnerships / data sources: if you have official or primary sources we should reference.
  • Technical issues: layout bugs, accessibility issues, or page errors.

How we handle corrections

When you report an issue, we verify against primary sources where possible (e.g., CPF, IRAS, HDB, MAS/LTA) and ship a versioned update. If a change is material, we add a note to the page.

Before you email

  • Paste the page URL you’re referring to.
  • Tell us the numbers you entered (or attach a screenshot of inputs/outputs).
  • If you’re referencing a rule/rate, include the source link.
