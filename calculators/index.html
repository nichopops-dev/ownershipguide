How to use these calculators (without fooling yourself)

  1. Choose the right comparison first. If you’re comparing “monthly instalment”, you’re already losing. Use Decision Comparisons.
  2. Run the calculator with conservative inputs. Assume higher maintenance, higher rates, and slower income growth.
  3. Stress-test fragility. If the only way the decision “works” is under perfect conditions, it’s not a plan — it’s a bet.
